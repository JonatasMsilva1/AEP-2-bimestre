Smart Urban

O Smart Urban é um sistema desenvolvido como projeto da AEP (Atividade de Estudo Programada) do curso de Engenharia de Software. O objetivo da aplicação é facilitar a comunicação entre os cidadãos e os órgãos responsáveis pela gestão urbana, permitindo o registro e acompanhamento de ocorrências relacionadas à cidade.

Funcionalidades
Cadastro e gerenciamento de ocorrências urbanas;
Registro de problemas como iluminação pública, descarte irregular de lixo e mobilidade urbana;
Interface simples e intuitiva;
Acompanhamento do status das solicitações;
Organização das informações para auxiliar na tomada de decisões pelos órgãos competentes.
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Git e GitHub
Objetivo do Projeto

O projeto busca demonstrar como a tecnologia pode contribuir para a construção de cidades mais inteligentes, promovendo maior participação da população e auxiliando na identificação e resolução de problemas urbanos.

Equipe

Projeto desenvolvido como atividade acadêmica para a disciplina de Engenharia de Software da Unicesumar.
